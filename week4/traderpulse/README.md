# TraderPulse

**TraderPulse** is a high-performance, AI-powered real-time market analysis dashboard. It combines live market data from Yahoo Finance with sentiment analysis powered by Gemini 3 Flash.

## Repository Structure

This repository is a monorepo containing both the backend service and the frontend application.

- **[Backend](./backend/README.md)** (`/backend`): FastAPI service handling data aggregation, AI integration, and API endpoints.
- **[Frontend](./frontend/README.md)** (`/frontend`): Next.js 16 application providing the user interface, charts, and dashboards.

## Architecture

TraderPulse uses a decoupled architecture to ensure scalability and maintainability.

> See **[ARCHITECTURE.md](./ARCHITECTURE.md)** for detailed system diagrams and design decisions.

## Key Features

- **Real-Time Data**: Live ticker and charts powered by Yahoo Finance.
- **AI Sentiment**: "Bullish/Bearish" indicators generated by Google Gemini.
- **Modern UI**: Built with Shadcn/UI and Tailwind CSS for a premium feel.
- **Docker Ready**: Full stack deployable via Docker Compose.
